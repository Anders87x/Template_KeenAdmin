Quick Start

The steps below for the default version (with build tools). If you want to use classic version (without build tools), go to this link instead.

  1. Download the latest theme source from the marketplace.

  2. Download and install Node.js from nodejs.org/en/download/

  3. Start command prompt window or terminal and change directory to [root]/theme/default/tools/

    cd [root]/theme/default/tools

  4. Install the latest version of npm.

    npm install --global npm@latest

  5. Install the latest version of yarn.

    npm install --global yarn

  6. Install gulp.

    npm install --global gulp-cli

  7. Install yarn dependencies. Must execute in [root]/theme/default/tools/ folder.

    yarn

  8. Compile the assets using gulp after yarn command.

    gulp

  9. Start the localhost server.

    gulp localhost

    Keep the console open. Open this link to run http://localhost:8080/default/

  10. Compile the edited assets (sass and js source files)

    gulp

    For more information about the Build Tools, click here.

    By default assets folder does not exist and it will be generated only after running gulp task.

  11. This is an optional gulp task for demo purpose. It does set all the demo backend URL to point to demo API in server.

    gulp apiurl

    For more information about the Build Tools, click here.

  12. The customization should be done from [root]/theme/default/src. After recompiling, use the assets found in [root]/theme/default/dist/[demo_id]/assets.

  13. Watch the edited assets files (sass and js source files)

    gulp watch

    Watcher tracks changes to the files and automatically runs the compiler.

  14. Watch the edited sass files

    gulp watch:scss

    Run the watcher for the SASS files only.

  15. Watch the edited js files

    gulp watch:js

    Run the watcher for the JS files only.
